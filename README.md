# Kristie's Blog

A personal learning project built with Astro, based on the official Astro Docs tutorial: “Build Your First Astro Blog”. This project explores how Astro works by building a complete blog from scratch through to deployment.

## Overview

This tutorial-based project demonstrates how to build a fully functioning blog using Astro, covering key concepts such as routing, components, and static site generation.

## What This Project Covers

- Setting up a full development environment for Astro
- Creating pages and blog posts for a blog website
- Building reusable UI using Astro components
- Querying and working with local files (content collections)
- Adding interactivity to the site where needed
- Preparing the project for deployment
- Deploying the final website to the web

## What I Learned

This project helped me understand Astro’s core features, including file-based routing, component-based architecture, and working with local content to generate dynamic pages.

## Outcome

A fully functioning blog built with Astro, ready for deployment and further extension.
